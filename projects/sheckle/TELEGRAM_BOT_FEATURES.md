## STANDARD TELEGRAM BOT FEATURES

### 1. Command Responses (/commands)
Users type commands, bot replies instantly:

**For SHECKLE:**
- `/price` - Current token price, market cap, 24h change
- `/chart` - Link to DexScreener chart
- `/ca` - Contract address (copy-paste ready)
- `/dashboard` - Link to whale watch dashboard
- `/holders` - Top 10 holder distribution
- `/buy` - Direct link to Pump.fun

**For SynSync:**
- `/protocols` - List of available protocols
- `/anxiety` - Quick link to anxiety relief protocol
- `/focus` - Quick link to focus protocol
- `/sleep` - Quick link to sleep protocol
- `/science` - Link to research citations
- `/app` - Link to SynSync Pro
