# WYRD Mini-App for Farcaster
## Beyond Word-a-Day: Wisdom Through Inversion

### The Problem with Current "Wisdom" Apps
- Word-a-Day teaches vocabulary, not discernment
- Knowledge without application is just trivia
- No framework for breaking societal spells
- No mechanism for taking power back

### WYRD: The Inversion Framework

**Core Philosophy:** The truth is usually the inverse of what we're told.

```
Mainstream Narrative          WYRD Inversion
─────────────────            ───────────────
"Just work harder"           → "Work smarter, own the means"
"Trust the experts"          → "Verify everything, trust yourself"
"Consume to be happy"        → "Create to be fulfilled"
"Follow the crowd"           → "The crowd is manipulated"
"Security comes from jobs"   → "Security comes from skills"
```

### Daily WYRD Structure

**Morning Inversion (The Spell Breaker)**
```
┌─────────────────────────────────────┐
│  Today's Inversion                  │
│                                     │
│  "They say: [Mainstream narrative]" │
│                                     │
│  "WYRD says: [The inverse truth]"   │
│                                     │
│  [Reflect] [Share] [Apply]          │
└─────────────────────────────────────┘
```

**Daily Components:**
1. **The Inversion** - One mainstream belief, flipped
2. **The Evidence** - Historical/empirical support for the inverse
3. **The Application** - How to apply this today
4. **The Community** - Others' applications and results
5. **The Streak** - Days of continuous spell-breaking
