## COMPOSITE STRATEGY

### Method A: Sequential Alpha Blending (Photoshop/GIMP)
1. Start with Layer 7 (Alex Grey) at 100%
2. Add Layer 6 (McFarlane) at 70% multiply blend
3. Add Layer 5 (Anime) at 60% screen blend
4. Add Layer 4 (South Park) at 50% normal blend with mask
5. Add Layer 3 (Virtua Fighter) at 40% overlay
6. Add Layer 2 (Starfox) at 30% add blend
7. Add Layer 1 (ASCII Key) at center, 100% with outer glow

### Method B: AI Image-to-Image Chaining
1. Generate Layer 7 base
2. Use as img2img input for Layer 6 at 0.6 strength
3. Continue down, decreasing strength each layer
4. Final pass: img2img with ASCII Key at 0.9 strength center crop

### Method C: Manual Collage + AI Upscaling
1. Generate each layer separately
2. Composite manually (precise control)
3. Run final composite through AI upscaler
4. Add film grain/unification pass
