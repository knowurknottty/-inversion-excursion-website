## Steganography Technical Details

**Method:** LSB (Least Significant Bit) in PNG RGB channels
**Capacity:** ~3 bits per pixel (1 bit per RGB channel)
**Image Size:** ~500x500 pixels = ~250,000 pixels = ~750,000 bits = ~93KB max payload
**Actual Payload:** 438 characters = ~3,500 bits (well within capacity)
**Detection Resistance:** High - requires specific tools to detect
**Extraction:** Simple Python script using PIL/numpy

**Security Note:** This is "security through obscurity" - not encryption, just hiding. Anyone with knowledge of steganography can extract. Purpose is recruitment filter, not secure communication.
