### [LEARNED: infrastructure]
- Auto-commit is non-negotiable. Lost work is unacceptable.
- Multiple fallback paths for every external API (PubMed → Europe PMC → Jina AI)
- Browser automation requires --headless=new on VPS; --no-sandbox mandatory for root

### [LEARNED: research-intelligence]
- DOI verification: curl -I https://doi.org/[DOI] — 302 = valid
- Europe PMC as PubMed fallback: faster, less rate-limiting
- Nature DOIs: check exact format (common typo: 01290-3 vs 01270-2)
- Always verify negative results — they're harder to publish, more valuable

### [LEARNED: content-production]
- Lead with controversy — builds credibility
- Cite DOIs explicitly — differentiates from competitors
- Acknowledge negative results — builds trust
- Never claim cure/disease modification — regulatory/credibility risk
