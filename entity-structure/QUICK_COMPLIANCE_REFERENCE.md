## DO's and DON'Ts

### ✅ ALWAYS DO

- Emphasize gameplay utility
- Highlight skill-based rewards
- Use tokens immediately upon acquisition
- Frame staking as "security deposit for service"
- Describe validator rewards as "accuracy bonuses"
- Call battle prizes "winnings" not "returns"
- Add disclaimer: "Tokens for gameplay, not investment"

### ❌ NEVER DO

- Promise price appreciation
- Suggest passive income
- Compare to investment products
- Use terms like "HODL," "moon," "diamond hands"
- Predict future token value
- Show price charts as "growth"
- Use APY/yield terminology

## FAQ FOR USERS

### Q: Is $EPWORLD an investment?
**A:** No. $EPWORLD is a utility token for gameplay. Use it for battles, validation, and research. It's not an investment and doesn't provide any expectation of profit.

### Q: How do I earn rewards?
**A:** Rewards are earned through skill-based activities: winning battles, accurate document validation, completing research bounties, and sustained gameplay focus.

### Q: What if the token price goes down?
**A:** Token value is based on utility for gameplay. Acquire tokens only if you intend to use them within the platform. We do not make any promises about token price.

### Q: Can I stake for passive income?
**A:** "Staking" in EPWORLD is a security deposit for validation service. Rewards are based on validation accuracy (skill), not time held.

### Q: Should I buy tokens now to sell later?
**A:** $EPWORLD tokens are designed for consumptive use, not speculation. Acquire tokens only if you plan to use them for gameplay.
