# Inversion Excursion - Image Generation Prompts for Grok Imagine

## Style Guide for All Images
**Base style keywords to include in every prompt:**
- Dark fantasy art style, mystical atmosphere
- Deep shadows with golden/amber accent lighting
- Ancient meets futuristic aesthetic
- Rich textures, detailed linework
- Cinematic composition
- Color palette: Deep blacks, midnight blues, warm golds, ember oranges

## Usage Notes

1. **For each chapter:** Select 2-3 images that best represent the core concepts
2. **Aspect ratios:** 
   - Hero images: 16:9 or 21:9 widescreen
   - Chapter headers: 16:9
   - Card backgrounds: 4:3 or 1:1
   - Full page illustrations: 9:16 or 2:3

3. **Color consistency:** Maintain the dark background + golden accent theme across all images
4. **Mood consistency:** Mysterious, empowering, ancient-meets-futuristic
5. **Text safety zones:** Keep important visual elements away from where text overlays will go
