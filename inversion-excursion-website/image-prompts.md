## Style Guide & Visual Identity

### Core Aesthetic: "Ancient-Futurist Dark Fantasy"

**Visual References:**
- **Zdzisław Beksiński** - Organic horror, dreamlike textures
- **H.R. Giger** - Biomechanical fusion, dark elegance
- **Roger Dean** (Yes album covers) - Surreal landscapes, otherworldly architecture
- **Kay Nielsen** - Art Nouveau fantasy, intricate linework
- **Moebius (Jean Giraud)** - Clean lines, cosmic scope

**Color Palette:**
| Role | Hex | Usage |
|------|-----|-------|
| Deep Void | #0a0a0f | Primary backgrounds |
| Midnight Blue | #12121a | Secondary backgrounds |
| Aged Parchment | #c9a227 | Accent gold, warmth |
| Ember Glow | #8b6914 | Secondary accent |
| Spirit White | #e8e8ec | Text, highlights |
| Ether Violet | #7c3aed | Crown chakra, mystical |
| Flame Gold | #f59e0b | Fire element, transformation |
| Water Silver | #94a3b8 | Water element, flow |

**Lighting Philosophy:**
- Chiaroscuro - dramatic light/dark contrast
- Single primary light source per image
- Volumetric fog/atmospheric depth
- Rim lighting on key figures
- Internal glow for mystical elements

## AI Generation Best Practices

### For Midjourney v6:
1. **Use --style raw** for more literal interpretations
2. **Use --ar for aspect ratios** matching target dimensions
3. **Use --q 2** for maximum quality on important images
4. **Negative prompts** via --no parameter for unwanted elements
5. **Reference styles** by naming artists (Beksiński, Giger, Moebius)
6. **Version control**: Save seeds for regeneration consistency

### For DALL-E 3:
1. **Be explicit about composition** (wide shot, close-up, profile)
2. **Specify lighting** (volumetric, rim light, internal glow)
3. **Mood descriptors** work well (mysterious, transcendent, oppressive)
4. **Iterate on rejects** with specific feedback
5. **Use GPT-4** to expand simple prompts into detailed versions

### For Stable Diffusion XL:
1. **Use quality tags**: "masterpiece, best quality, 8k uhd"
2. **Negative prompts are essential** for SD
3. **CFG scale 7-8** for good prompt adherence
4. **DPM++ 2M Karras** sampler for quality
5. **Hi-res fix** at 0.5 denoise for final upscale

### General Tips:
1. **Generate at 2x target size** then downscale for sharpness
2. **Batch generate** (4 variations) and select best
3. **Post-process** in Photoshop/GIMP for color correction
4. **Convert to WebP** for 30-50% smaller file sizes
5. **Test at multiple sizes** to ensure legibility at small scales
