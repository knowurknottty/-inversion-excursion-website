## 6. Keyword Density Guidelines

### Recommended Density by Element
| Element | Primary Keyword | Secondary Keywords | LSI Terms |
|---------|----------------|-------------------|-----------|
| Title Tag | 1 (exact match) | 1-2 variations | - |
| H1 | 1 (exact or partial) | - | - |
| First 100 words | 1-2 | 2-3 | 3-5 |
| Body content (per 500 words) | 1-2 | 3-5 | 8-12 |
| H2-H6 | Include where natural | 1 per section | - |
| Image alt text | 1 if relevant | - | - |
| Meta description | 1 (natural) | 1-2 | - |
