# Inversion Excursion - Alt Text Inventory

## Quick Reference for Accessibility

### Legend
- **Decorative**: Empty alt (`alt=""`) - screen readers skip
- **Informative**: Descriptive alt text - conveys meaning
- **Complex**: Long description needed (figcaption or aria-describedby)

---

| Element | Type | Alt Text |

| Logo/Favicon | Decorative | `alt=""` (accompanied by text title) |

## SEO-Optimized Alt Text Guidelines

### Primary Keywords to Include Naturally:
- "Inversion Excursion"
- "Frequency Warrior"
- "mudrā" / "mudra"
- "elemental keys"
- "Seven Dungeons"
- "consciousness"
- "liberation"

### Writing Tips:
1. **Front-load important words** - "The Pedant boss..." not "An image of..."
2. **Be specific** - "golden dollar sign eyes" not "scary eyes"
3. **Include emotional context** - "representing..." 
4. **Keep under 150 characters** for simple images
5. **Use longer descriptions** (200-250 chars) for complex illustrations
6. **Never keyword stuff** - keep it natural and descriptive

### Examples of SEO + Accessibility Balance:

**Good:**
```html
alt="The Pedant—boss of the Ivory Tower in Inversion Excursion, 
     a figure fused with bookshelves representing credentialism"
```

**Too keyword-stuffed:**
```html
alt="Inversion Excursion mudra yoga meditation frequency warrior 
     consciousness liberation book chapter"
```

**Too vague:**
```html
alt="A picture of a boss"
```

## Decorative Images (Empty Alt)

The following should have `alt=""`:

1. **CSS-generated patterns/gradients**
2. **Icons that accompany text** (the text describes them)
3. **Borders and separators**
4. **Decorative flourishes**
5. **Background images**
6. **Repeated logos in footers** (if main logo is described)

---

## Implementation Checklist

- [ ] All informative images have descriptive alt text
- [ ] All decorative images have empty alt (`alt=""`)
- [ ] No "image of" or "picture of" prefixes
- [ ] Complex images have figcaption or aria-describedby
- [ ] Keywords included naturally for SEO
- [ ] Character limits observed (150 simple, 250 complex)
- [ ] Tested with screen reader (NVDA/VoiceOver)
