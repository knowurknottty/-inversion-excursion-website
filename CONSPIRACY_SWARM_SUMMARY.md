## SIDE QUESTS (7-15)

### Quest 7: "TESTIMONY OF THE DAMNED - PART I: SARAH"
- **Survivor testimony series opener**
- Boss: Agent Krol (retrospective cleaner, 100 HP)
- Choice: Public testimony / Hidden identity / Anonymous
- Unlocks: Survivor network, corroborating evidence

### Quest 8: "TESTIMONY OF THE DAMNED - PART II: MARCUS"
- **Pilot witness, alcoholic**
- Boss: Addiction/Trauma (environmental/social challenge)
- Choice: Rush drunk testimony / Recovery investment / Unauthorized use
- Unlocks: Complete flight manifests

### Quest 9: "TESTIMONY OF THE DAMNED - PART III: THE COLLECTIVE"
- **Survivor network finale**
- Boss: Mother Eve (social/charismatic, no combat)
- Choice: Full alliance / Limited partnership / Refuse / Expose
- Unlocks: Maximum resources vs. integrity path

### Quest 10: "PLANTED EVIDENCE"
- **Infiltration/planting quest**
- Boss: None (pure stealth)
- Choice: Real evidence / Fabricated / Refuse
- Unlocks: Corrupt prosecutor's files

### Quest 11: "THE PENTHOUSE INFILTRATION"
- **High-society gala heist**
- Boss: Victor Lazarus (keeper, 125 HP, persuadable)
- Choice: Violent interrogation / Peaceful persuasion / Blackmail
- Unlocks: Original blackmail archive location

### Quest 12: "THE SERVER FARM"
- **Technical infiltration**
- Boss: Daemon (system admin, technical duel)
- Choice: Full download / Selective / Tip off owner
- Unlocks: Redundant Phoenix data

### Quest 13: "THE FORGER'S CLIENTS"
- **Sienna Vance follow-up**

- Choice: Kill / Pay off / Recruit
- Unlocks: Complete client list (dozens of leads)

### Quest 14: "THE JUDGE'S SECRET"
- **Judicial infiltration/blackmail**
- Boss: None (political/social)
- Choice: Blackmail for advantage / Public exposure / Anonymous tip
- Unlocks: Judicial influence or reform

### Quest 15: "THE WHISTLEBLOWER'S GAMBIT"
- **Finale tie-in extraction**
- Boss: None (timed extraction)
- Choice: Extract (risk operation) / Refuse (preserve plan) / Fake attempt
- Unlocks: Authentic intel for final quest

## MORAL CHOICE ARCHITECTURE

### Choice Categories:
1. **Individual vs. Collective** (Quests 1, 3, 7, 8, 15)
2. **Truth vs. Protection** (Quests 4, 5, 14)
3. **Justice vs. Pragmatism** (Quests 2, 9, 10, 11)
4. **Means vs. Ends** (Quests 12, 13)
5. **Systemic Philosophy** (Quest 6)

### Consequence Tracking:
- **Humanity Score** (compassion vs. pragmatism)
- **Information Score** (evidence gathered vs. destroyed)
- **Resourcefulness Score** (creative solutions)
- **Reputation** (various factions)

## DESIGN PHILOSOPHY

### Core Tenets:
1. **Every quest advances multiple narratives** (main plot, character arcs, world-building)
2. **No purely optional content** — side quests affect main quest outcomes
3. **Moral choices have no "correct" answer** — only consequences
4. **Document pages are tangible** — physical objects players collect and review
5. **Bosses represent ideas** — each embodies a theme of the quest

### Player Arc:
- **Quests 1-2:** Learning the ropes, discovering the conspiracy
- **Quests 3-4:** Uncovering scale, encountering moral complexity
- **Quests 5-6:** Confronting the system, making defining choices
- **Side Quests:** Fleshing out the world, earning resources, testing ethics
