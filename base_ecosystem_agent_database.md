# Base Ecosystem Agent Activity Database

## Executive Summary

This database documents AI agent activity within the Base ecosystem as of March 2026. Base, incubated by Coinbase, has emerged as a major hub for autonomous AI agents with significant infrastructure support through Coinbase Developer Platform (CDP) AgentKit.

---

## 1. Agent Accounts & Infrastructure

### Primary Agent Frameworks

#### CDP AgentKit (Coinbase Developer Platform)
- **Official Toolkit**: `github.com/coinbase/agentkit`
- **Tagline**: "Every AI Agent deserves a wallet"
- **Status**: Actively maintained with nightly builds
- **Languages**: TypeScript/Node.js 22+, Python 3.10+
- **Networks**: Base, Base Sepolia, all EVM chains, Solana

**Key Features:**
- Secure wallet management for AI agents
- Onchain actions (transfers, swaps, smart contract deployments)
- Framework-agnostic (works with LangChain, Eliza, Vercel AI SDK, etc.)
- Multi-network support (EVM + Solana)

**Installation:**
```bash
# TypeScript
npm create onchain-agent@latest

# Python
pipx run create-onchain-agent
```

#### Eliza Framework (elizaOS)
- **Website**: `elizaos.ai`
- **Description**: "Your Agentic Operating System"
- **Use Cases**: AI Influencer, Crypto Trader, Customer Support, DAO Governance, Growth Hacker
- **Architecture**: Unified Message Bus, Composable Swarms, Strategic Action Chaining

**Key Partners:**
- Stanford's Future of Digital Currency Initiative
- Chainlink (CCIP and oracle network)
- Doodles (Dreamnet metaverse)

#### LangChain Integration
- Full-featured framework with extensive integrations
- CDP AgentKit provides LangChain extensions
- Best for complex agents requiring custom tools

#### Other Supported Frameworks
- Vercel AI SDK
- OpenAI Agents SDK
- Pydantic AI
- Strands Agents
- Autogen
- Model Context Protocol (MCP)
