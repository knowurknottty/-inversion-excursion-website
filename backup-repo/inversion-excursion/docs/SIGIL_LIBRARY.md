### Hiding Techniques

**Transparency:** 5-20% opacity for subtle presence
**Blending Modes:** Overlay, Soft Light, Multiply
**Size:** Micro-sigils (5-10px) to large (full page)
**Color:** Match surrounding palette, slight glow
**Distortion:** Slight warping to follow image contours

### Activation Triggers

**Visual:** Eye contact, prolonged viewing
**Physical:** UV light, magnification, specific angles
**Digital:** Hover, click, zoom
**Mental:** Meditation, frequency entrainment

### Safety Considerations

**Do NOT:**
- Overload single images with too many sigils
- Use triggering symbols without context
- Make hidden elements too disturbing
- Forget accessibility (alt text, descriptions)

**DO:**
- Layer meaning progressively
- Allow discovery at different levels
- Maintain aesthetic coherence
- Test with actual viewers
