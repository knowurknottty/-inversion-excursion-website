## POST-PROCESSING WORKFLOW

### Standard Layer Stack (Photoshop/GIMP)

```
Layer 10: Metadata/Export Info (hidden)
Layer 9:  Color Lookup Table (optional grading)
Layer 8:  Vignette (darken edges)
Layer 7:  Texture Overlay (paper grain 15-25%)
Layer 6:  Sigil Overlay Group (sigils with blend modes)
  - Sub-layer: Individual sigils at 5-15% opacity
  - Blend mode: Overlay or Soft Light
Layer 5:  Light Effects (glows, rays)
Layer 4:  Color Correction (Curves/Levels)
Layer 3:  Base Image (AI generation)
Layer 2:  Background Texture (if needed)
Layer 1:  Solid Color Base (locked)
```

### Sigil Embedding Steps

1. **Import sigil** as separate layer
2. **Position** according to Hidden Elements List
3. **Scale** appropriately (usually 2-5% of image size for micro-sigils)
4. **Set blend mode** to Overlay, Soft Light, or Screen
5. **Adjust opacity** to 5-15% (refer to table)
6. **Apply color** matching adjacent tones
7. **Add subtle distortion** if needed (Filter > Distort > Wave/Ripple)
8. **Merge or group** when satisfied

### Export Checklist

- [ ] All sigils embedded at correct opacity
- [ ] Color correction applied
- [ ] Sharpening applied (Web: unsharp mask 50-100%, Print: 150-200%)
- [ ] Metadata embedded (JSON format)
- [ ] Color space verified (sRGB for web, CMYK for print)
- [ ] Resolution verified
- [ ] File format appropriate for use case
- [ ] Backup of layered master file saved

```bash
# Add sigil overlay with transparency
```
