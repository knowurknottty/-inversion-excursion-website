## COMPOSITE STRATEGY

### Layer Blending Modes (Photoshop/GIMP/Blender)

| Layer | Blend Mode | Opacity | Notes |
|-------|-----------|---------|-------|
| 7 (Alex Grey) | Normal | 100% | Base cosmic aura |
| 6 (McFarlane) | Multiply | 85% | Darkens, adds depth |
| 5 (Anime) | Screen | 70% | Adds energy, motion |
| 4 (South Park) | Overlay | 60% | Texture injection |
| 3 (Virtua Fighter) | Soft Light | 75% | Retro gaming vibe |
| 2 (Starfox) | Color Dodge | 50% | Wireframe highlights |
| 1 (ASCII) | Normal | 90% | Core text layer |
| Center (Key) | Add/Glow | 100% | Focal point |

### Post-Processing Stack

1. **Chromatic Aberration** - +5px RGB split (retro gaming feel)
2. **Vignette** - Heavy, darkening corners 40%
3. **Film Grain** - 35mm, medium intensity
4. **Scanlines** - Subtle, 20% opacity
5. **Glow/Bloom** - Center-focused, high intensity
6. **Color Grading** - Lift shadows to blue, push highlights to gold

### Technical Specifications

- **Final Resolution:** 3000x3000px (album cover standard)
- **Color Space:** sRGB for web, CMYK conversion for print
- **Format:** PNG (master), JPG (web), PSD (layered)
- **Bleed:** 150px on all sides for print

## GENERATION WORKFLOW

### Phase 1: Individual Layer Generation
1. Generate each layer separately using prompts above
2. Export at 3000x3000px minimum
3. Save as PNG with transparency where needed

### Phase 2: Compositing
1. Import all layers into Photoshop/GIMP/Blender
2. Apply blend modes per table above
3. Adjust opacity and layer masks
4. Add The Inversion Key at center

### Phase 3: Post-Processing
1. Apply post-processing stack
2. Color grade for cohesion
3. Add final vignette and grain
4. Export final versions

### Phase 4: Variations
1. Create "clean" version (no scanlines/grain)
2. Create "vintage" version (heavy degradation)
3. Create "minimal" version (layers 1, 2, 4, Center only)
