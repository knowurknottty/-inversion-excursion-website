## KEY COMBOS

### High Damage
```
Beta → Gamma → 432Hz (Harmonic Burst chain)
└── Critical strike for 150%+ damage
```

### Mass Healing
```
Delta → Theta → Schumann
└── Recovery effects +25%, shields last longer
```

### Control Lock
```
Pedant P-4 (Cartesian Prison) → Puppeteer PP-4 (Grand Theatre)
└── Lock frequency + control opponent's hand
```

### Mill Protection
```
Alchemist A-3 (Philosopher's Stone) after Feed F-3 (Feeding Frenzy)
└── Turn forced discard into HP gain
```

### Boss Killer
```
Alchemist A-5 (Great Work Completed) at full shields
└── Convert all shield to damage (100+ damage possible)
```

## GAME FLOW

```
TURN STRUCTURE:
1. Draw Phase      → Draw 1 card (or more with effects)
2. Energy Phase    → Gain 1 energy per turn (max 10)
3. Main Phase      → Play cards, activate abilities
4. Combat Phase    → Resolve attacks
5. End Phase       → Check win/loss conditions

WIN CONDITIONS:
├── Reduce opponent HP to 0
├── Opponent cannot draw (mill)
├── Special card effects (A-5, F-5)
└── Boss phase transitions

LOSS CONDITIONS:
├── Your HP reduced to 0
├── Cannot draw when required
├── Boss enrage timer expires
└── Special curse effects (I-5 draw skip)
```

## KEY TERMS GLOSSARY

| Term | Definition |
|------|------------|
| **Inversion** | Card mechanic where every power has a curse |
| **Frequency** | Brainwave state alignment (Delta→Gamma, Schumann, 432Hz) |
| **Resonance** | Positive interaction between compatible frequencies |
| **Discord** | Negative interaction between incompatible frequencies |
| **Curse** | Negative side effect triggered by card play |
| **Tier** | Rarity/power level (Physical→Cosmic) |
| **Essence** | Crafting currency earned through play |
| **Mill** | Strategy of forcing opponent to discard/run out of cards |
| **OTK** | One Turn Kill — winning in a single turn |
| **Token** | AI-controlled creature that attacks/defends |
| **Lock** | Restriction preventing certain card plays |
| **OTB** | Out The Box — unconventional deck strategies |
